# Инструкция по установке и использованию GUI-моста Diaphora MCP

GUI-мост позволяет MCP-серверу выполнять экспорт баз данных мгновенно прямо из запущенной и открытой на экране программы IDA Pro, избегая конфликтов блокировки файлов и необходимости закрывать IDA.

---

> [!NOTE]
> **Важное обновление**: Устаревший XML-RPC мост (`diaphora_gui_listener.py` на порту `28652`) был **объявлен устаревшим и удален** из-за ограничений потокобезопасности UI и дедлоков в цикле обработки событий Qt.
> 
> Теперь интеграция работает через официальный надежный **HTTP-плагин `ida_mcp.py`** (по умолчанию запускается на порту `13337`).

## Шаг 1. Установка плагина в IDA Pro

Плагин является частью пакета `ida-pro-mcp`. Для его установки:
1. Найдите файл плагина `ida_mcp.py` в репозитории по пути `idalib/ida_mcp.py`.
2. Скопируйте его в директорию плагинов пользователя IDA Pro:
   - **Windows**: `%APPDATA%\Hex-Rays\IDA Pro\plugins\ida_mcp.py` (например, `C:\Users\<Имя>\AppData\Roaming\Hex-Rays\IDA Pro\plugins\ida_mcp.py`)
   - **macOS/Linux**: `~/.idapro/plugins/ida_mcp.py`

*Плагин автоматически загрузится при запуске GUI IDA Pro.*

---

## Шаг 2. Запуск и проверка работы

1. Откройте нужную базу данных (например, `sqlite3_aimp.dll.i64`) в GUI IDA Pro.
2. В меню IDA Pro выберите **Edit -> Plugins -> MCP** (или нажмите **Ctrl+Alt+M**), чтобы запустить HTTP-сервер интеграции.
3. В нижней панели Output Window в IDA Pro должны появиться следующие строки:
   ```
   [MCP] Plugin loaded, server will start automatically
   Config: http://127.0.0.1:13337/config.html
   Diaphora: http://127.0.0.1:13337/diaphora/health
   ```
4. Попросите ИИ-агента экспортировать базу данных:
   > *“Export sqlite3_aimp.dll.i64”*
5. Клиент автоматически обнаружит запущенную сессию GUI на порту `13337`, убедится, что запрашиваемый файл совпадает с открытым в GUI, и запустит фоновый экспорт в потокобезопасном воркере.
6. Вы увидите лог прогресса экспорта в консоли клиента в реальном времени:
   ```
   [Diaphora MCP] Export progress: 6% (200/3362 functions)...
   [Diaphora MCP] Export progress: 12% (400/3362 functions)...
   ```

---

## Параллельный headless-экспорт

Если вы запрашиваете экспорт базы данных, которая **не открыта в GUI в данный момент**, клиент автоматически проигнорирует GUI и запустит фоновый автономный процесс `idat.exe`.

При этом ваше активное GUI-окно останется полностью отзывчивым, и вы сможете продолжать работу без каких-либо задержек или зависаний.
